### Requirements

* Scala 2.11.7
* Java 1.8
* Play Framework 2.4.x

### Running

```sh
$ activator run
```
### Unit Tests and Integration Test

```sh
$ activator test
```

### Deployment

```sh
$ sbt stage deployHeroku
```
